There are nine superglobal variables in PHP. They are:

1. $_SERVER - Contains information about server and execution environment, such as headers, paths, and script locations.

2. $_GET - Contains variables passed to the current script via URL parameters.

3. $_POST - Contains variables passed to the current script via HTTP POST method.

4. $_FILES - Contains information about uploaded files via HTTP POST method.

5. $_COOKIE - Contains variables passed to the current script via HTTP Cookies.

6. $_SESSION - Contains variables stored in the user's session.

7. $_REQUEST - Contains the contents of $_GET, $_POST and $_COOKIE.

8. $_ENV - Contains variables passed to the current script via the environment.

9. $GLOBALS - Contains a reference to all variables which are currently defined in the global scope of the script, including those from the superglobals.
